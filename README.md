# Балансировка обратного одноосевого маятника

## Описание

Данный репозиторий посвящён проектированию, моделированию и изучению устройства балансировки обратного одноосевого маятника (инвертированного маятника на одной оси).

## Цель проекта

Разработка и исследование системы стабилизации обратного одноосевого маятника в вертикальном положении с применением современных методов управления.

## Задачи

- Математическое моделирование динамики маятника
- Проектирование регулятора (PID, LQR и др.)
- Создание симуляции работы системы
- Анализ устойчивости и качества управления
